# Decitizen

## Goal
A way to validate a Kenyan using ID, First Name, and Year of birth.

## Requirements
- Node or Bun

## Building
- Clone This repo
- Enter the directory you cloned into
```bash
cd Decitizen
```

- Install the libraries
```bash
npm i
```

- Run (Currently server not implemented yet)
```bash
npm run server
```
